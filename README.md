# Data Structures In C++

This repository holds struct & methods of different data structures like:
* [Stack](#stack)
* [Queue](#queue)
* [Binary Search Tree](#binary-search-tree)

## Stack

Methods:
* push(data: T&) -> void
* pop() -> void
* peek() -> T
* display() -> void

## Queue

Methods:
* enque(data: T&) -> void
* deque() -> bool, T
* display() -> void

## Binary Search Tree

Methods:
* insert(data: T) -> void
* remove(target: T) -> void
* search(target: T) -> bool
* traversal(order: unsigned) -> void
